Home Linux Fileserver

Home file server (Debian) for storage and backups

Built a PC entirely from really old and discarded legacy hardware and used it as a personal home file server for storage, backups, and as a lab environment.

PC Specs (Super high tech):

  • CPU: AMD Phenom II X6 1045T (6-core)
  • Motherboard: Asus M5A78L-M LX PLUS
  • RAM: 8GB DDR3, 1333 MHz (2x 4GB)
  • SDD: 120GB Lexar SSD, 240GB ADATA SU630
  • HDD: 500GB Hitachi HDP72505, 1TB Hitachi HDS72101

Partitioning

  • SSD1 (120GB): /boot, swap, /
  • SSD2 (240GB): /home
  • HDD1 (500GB): /mnt/storage1
  • HDD2 (1TB): /mnt/storage2

Networking, Security:

  • nftables rules: default drop, allow-by-policy, allow SSH from personal computer IP address only
  • Disable remote root login
  • Disable password authentication (key only)

File Sharing

  • Automount on personal computer (Pop_OS! Linux) using systemd user .service unit and sshfs mount script
  • Samba access (mnt/storage1/smbshare) for Windows 10 access & file storage
